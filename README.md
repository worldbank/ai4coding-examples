# ai2coding: examples

This repository contains the source code and example materials that supplement the main course content for **AI for Coding**.

For the full course materials, see the main repository:

- [github.com/worldbank/ai4coding](https://github.com/worldbank/ai4coding)

Use this repository alongside the main course repository for hands-on examples, exercises, and supporting code.
